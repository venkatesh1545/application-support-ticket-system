# Application Support & Maintenance Ticket System

A Python-based console application that simulates real-world application support and maintenance workflows.

## Features
- Create support tickets
- Track ticket status (Open, In Progress, Resolved)
- Update and resolve tickets
- Store and retrieve data using SQLite
- Input validation and error handling

## Tech Stack
- Python
- SQLite
- Git & GitHub

## How to Run
```bash
python database.py
python main.py
```

## Purpose
This project demonstrates application support, maintenance, debugging, and analytical problem-solving skills aligned with Associate Software Engineer roles.
